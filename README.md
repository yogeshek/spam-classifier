  # Spam Classifier

  Binary classification ML model to detect spam messages using SMS Spam Collection dataset.

  ## Setup

  1. Clone the repository
  2. Create virtual environment: `python -m venv venv`
  3. Activate: `source venv/Scripts/activate`
  4. Install dependencies: `pip install -r requirements.txt`
  5. Download dataset and place in `data/` folder

  ## Usage

  ```bash
  # Train model
  python train.py

  # Make predictions
  python predict.py

  Dataset

  SMS Spam Collection from UCI ML Repository
